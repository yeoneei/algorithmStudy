# Linked List

## Linked List 구성요소
- 노드(Node)
    - 실제 정보를 담고 있는 데이터
- 링크(Link)
    - 노드의 연결을 나타내는 포인터 
```
class Node{
    T data;
    Node *pNext;
}
```

## Linked List의 특징
- 동적인 자료구조
    - 필요할때 할당, 필요없으면 해제
    - 동적인 메모리사용으로 미리 크기를 지정할 필요가 없음
- 불연속적인 메모리 사용
    - 물리적 순서 없고 불연속적으로 저장됨
    - 링크에 의해 순서가 유지됨
- 순차적 접근만 가능
    - 임의 접근 불가능
